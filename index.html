Deployment Module Project

  • Links to repos: UI Repository    Backend Repository    Infrastructure Repository
  • Presentation recording: Watch Presentation
  • Deployed App: Link
  • For this project, I worked solo to rebuild an earlier app that I previously couldn't deploy, aiming to succeed this time by using my new HP tech stack: Angular for the front end, .NET Core for the API, Docker, and Oracle Cloud Infrastructure (OCI) for hosting and deployment because it was the only applicable free solution I could find.
  • The app is a web-based Christmas Light Install quote generator and scheduler that I built for a friend who runs a Christmas light installation business. It allows users to log in, create quotes for customers, and view scheduled installations on a calendar. The coolest feature of the app is that it integrates with the Google Maps/Google Earth API to allow the user to remotely measure a roofline to understand the length of lighting needed without needing to visit the site until installation day.
  • Being that it was a solo project, I was responsible for everything, including starting over on app development (front end and back end), configuring CI/CD pipelines, setting up cloud resources, and ensuring the app worked in production.
  • Since it was not possible to teach us everything we needed to know in the deployment module, I had to independently research deployment strategies and troubleshoot issues on my own.
  • This project demonstrates my ability to learn new concepts quickly, solve problems independently, and see a project through to deployment.
  • Overcoming the challenges of deployment gave me confidence in my ability to deliver working software and adapt to new technical requirements.
